# Md. Saif Al Islam | Portfolio

This is a **personal portfolio website** of Md. Saif Al Islam, built using **HTML, Tailwind CSS, and JavaScript**. The website showcases personal information, skills, projects, and contact details.

<strong> Websitelink : </strong> <a href="https://ofear3.github.io/mohammad_saif-portfolio/">  <i> saif007.com  </i>  </a>

## Features

- **Responsive Design:** Works on mobile, tablet, laptop, and large screens.
- **Navigation:** Fixed top navigation for desktop and right-side slide-out menu for mobile.
- **Sections:**
  - Home
  - About Me
  - Skills
  - Projects
  - Contact Me
- **Projects:** Each project card has hover effects and shows used technologies.
- **Contact Form:** Users can send messages.
- **Footer:** Social links (Facebook, GitHub, LinkedIn) and copyright info.

## Tech Stack

- **HTML** - Markup and structure
- **Tailwind CSS** - Styling and responsiveness
- **JavaScript** - Mobile menu toggle & textarea auto-resize

## How to Use

1. Clone this repository.  
2. Open `index.html` in any browser.
